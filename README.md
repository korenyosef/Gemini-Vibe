# 🎮 Gemini Vibe

**Gemini Vibe** is a Chrome extension designed to make waiting for AI responses more entertaining. It automatically injects a sleek, customizable game overlay into the Google Gemini interface whenever the AI is "thinking" or generating a response.

## ✨ Features

- **🚀 Automatic Activation:** The game overlay appears instantly when you send a prompt and Gemini begins generating.
- **⏱️ Thinking Timer:** Tracks exactly how long Gemini has been processing your request.
- **🌈 Dynamic Glow:** The overlay border and status indicators change color to match your selected game.
- **🎮 Game Presets:** Choose from classic favorites:
  - 🕹️ **Pro Tetris**
  - 🔢 **2048 Master**
  - 🐍 **Retro Snake**
  - 🍒 **Classic Pacman**
  - 🏃 **T-Rex Runner**
- **🎨 Custom Games:** Enter any URL to play your favorite web-based game or tool while you wait.
- **🎛️ Settings Popup:** Easily switch games or set custom URLs through the extension's popup menu.
- **✖️ Manual Close:** Done playing before Gemini is? Use the quick-close button to dismiss the overlay.

## 🛠️ Installation

### For Developers / Manual Loading:

1. **Clone or Download** this repository.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click **"Load unpacked"**.
5. Select the folder containing the extension files.

## 📖 How to Use

1. Navigate to [Gemini](https://gemini.google.com/).
2. Click the **Gemini Vibe** icon in your extension toolbar to select your preferred game or enter a custom URL.
3. Type a prompt and hit **Enter** or click the **Send** button.
4. The game overlay will appear automatically while Gemini generates its response.
5. The overlay will disappear once the response is complete, or you can click the `✕` to close it manually.

## ⚙️ Customization

You can set a custom game URL via the extension popup:

1. Click the Gemini Vibe icon.
2. Enter a valid URL (e.g., `https://example.com/game`) in the "Custom Game URL" field.
3. Click **Set**.
4. Your custom game will now be the active "Vibe"!
